# A.8.1 Fusion 360 f3D Files for 3D Printing Model

This section contains the original **f3D design files** created in Autodesk
Fusion 360 for the 3D-printable components of the Crosier-Control DMI
(Digital Musical Instrument).

In addition to the CAD files, reference images and technical drawings are
included to support visualization, fabrication, and assembly processes.

## Contents

### Fusion 360 Design Files (f3D)
- `Crosier modelo v1.f3d` – Main structural model of the Crosier-Control instrument
- `Crosier Tapas smartphone.f3d` – Design file for the smartphone enclosure and covers

### Reference Images and Technical Drawings
- `C R O S I E R C O N T R O V2L.svg` – Vector graphic reference of the Crosier-Control design
- `planos final tapas smartphone.png` – Final technical drawing of smartphone covers
- `tapa superior planos (1).pdf` – Technical plans for the upper cover component

## Usage Notes
- The **f3D files** are intended to be opened and edited in Autodesk Fusion 360.
- Reference images (PNG, SVG, PDF) are provided for documentation and fabrication guidance.
- File names and dimensions should not be modified unless necessary for fabrication adjustments.

## Requirements
- Autodesk Fusion 360 or compatible CAD software
- Access to 3D printing or digital fabrication tools
